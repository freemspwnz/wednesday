# Changelog

## [Unreleased]

### Changed
- Проект переориентирован на новое ядро доменной модели.
- Выполнен полный отказ от прошлых наработок в core-части.
- Точка нового старта: `domain/user` и `domain/kernel`.

### Removed
- Устаревшие компоненты и архитектурные ветки, не соответствующие новой доменной стратегии.
- Наследие предыдущих итераций, мешающее целостности новой модели.

### Added
- Базовый доменный фундамент:
  - `domain/kernel` (базовые VO и доменные исключения),
  - `domain/user` (новая user-domain модель).
- Новый тестовый контур для домена:
  - `tests/domain/user`,
  - `tests/domain/kernel`.
- Обновленный quality gate для domain-слоя (mypy + pytest + coverage).

### Notes
- Этот релиз фиксирует архитектурный reset.
- Дальнейшее развитие ведется от чистой доменной базы `user + kernel`.
